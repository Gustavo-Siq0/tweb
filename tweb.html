módulos do curso

  1. Introdução ao HTML: Explicação básica sobre o que é HTML e sua importância na construção de páginas web.
  2. Estrutura do HTML: Apresenta a organização fundamental de um documento HTML, incluindo as principais tags como <html>, <head> e <body>.
  3. Títulos e Parágrafos: Ensina como utilizar as tags de título e parágrafo para organizar o conteúdo textual.
  4. Listas: Mostra como criar listas ordenadas , não ordenadas e listas de definição .
  5. Links e Imagens: Explica como adicionar links de navegação com a tag e como inserir imagens com a tag .
  6. Tags Estruturais: Introduz as tags semânticas como ,main,;section, articlee, footer para estruturar melhor o conteúdo da página.
  7. Formulários: Ensina como criar formulários com campos de entrada, botões, caixas de seleção, entre outros, usando a tag, form e seus elementos.
  8. CSS: Introdução ao CSS (Cascading Style Sheets), a linguagem usada para estilizar e dar aparência às páginas HTML.
